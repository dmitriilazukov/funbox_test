# Funbox.ru тестовое задание
Текст задания https://funbox.ru/q/python.pdf

## Инициализация
```
pip3 install pipenv
git clone https://github.com/dmitriilazukov/funbox_test.git
cd funbox_test
export PIPENV_VENV_IN_PROJECT=true
make init
echo DONE
```
## Запуск сервера
```
make start
```
## Запуск тестов
```
make test
```
